Usage

Android ProgressBar Circle Generator is online tool to generate custom ProgressBar circle for Android Apps.

Steps

# Customize circle in Properties panel.
# And copy source code.
# Create XML file in drawable folder and paste it.
# Change ProgressBar circle => android:indeterminateDrawable="@drawable/custom_circle"
# Thats it!
